Hallo mijn naam is Bas Vriezen, ik ben student aan de Haagse Hogeschool binenn de opleiding Communicatie en Multimedia Design, meestal afkekort als CMD. Voor het vak IxD & Technologie was de opdracht om een portfoliowebsite te maken.

Binnen deze site zijn een aantal opdrachten/ Challenges te zien met daarin een toepassing van Responsive Web Design. Responsive Web Design wil zeggen dat één website op meerdere devices werkt en voor elk device een aangepaste lay-out heeft. Doormiddel van de code kan een website zich dan aanpassen aan de schermgrootte van het device waarop het bekeken wordt. De lay-out voor de mobiel zal dus anders zijn dan op bijvoorbeeld een laptop. Het voordeel van Responsive Web Design is dat programmeurs geen aparte website hoeven te maken voor Smartphone, Tablet, Laptop en PC. Simpelweg door binnen de code aan te geven op welke schermbreedte de lay-out moet veranderen kunnen websites gemaakt worden voor alle soorten devices.
